Стандарт

За основу был взят стандарт CommonMark Spec и его диалект  GitHub Flavored Markdown.


Из CommonMark было взято - заголовки, списки разных видов, цитаты, ссылки.

Из GitHub Flavored Markdown было взято -  таблицы, зачеркнутый текст, списки задач с галочками, блоки кода с подсветкой.



Заголовки:

# Заголовок H1
## Заголовок H2
### Заголовок H3
#### Заголовок H4
##### Заголовок H5
###### Заголовок H6

---

Стили текста:

**Жирный текст** (двойные звездочки)
*Курсив*  (одна звездочка)
***жирный курсив*** (три звездочки)
~~Зачеркнутый текст~~ (две тильды)

2. Списки

Маркировочнные списки:
* элемент со звездочкой
* элемент 2
  * Вложенный элемент 1
    * Вложенный элемент 2

- элемент с дефисом
- элемент 2

+ элемент с плюсом
+ элемент 2





Нумерованные списки:
1. Первый
2. Второй
   1. Вложенный нумерованный 1
   2. Вложенный нумерованный 2
3. Третий
4) Четвертый



Списки задач:
- [x] Выполненная задача
- [ ] Невыполненная задача
- [ ] Еще одна задача
